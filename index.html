Escapade

I love to travel and was keen to try building my own flight search web app.

Using the kiwi.com flights search API, the app utilises a React front-end with CSS grid for styling and Node.js with Express for handling the API call.

I focussed on creating a mobile friendly search experience within a modern clean UI - built without libraries.

Visit escapade! GitHub repository

BeanCounter

Tracking expenses is hard, but the latest apps in fintech are making it easier. I wanted to try making my own expense tracker!

I practised my React & Redux skills whilst also incorporating Highcharts JS for visualising the data.

To provide data persistence, I integrated local storage functionality.

The app was built "mobile first" and utilises CSS grid and Sass for styling.
